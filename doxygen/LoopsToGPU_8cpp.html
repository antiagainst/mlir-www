template<typename OpTy >
static OpTy createGPULaunchLoops ( OpTy  rootForOp,
ArrayRef< Value >  ids,
ArrayRef< Value >  nids 
)
static

Given nDims perfectly nested loops rooted as rootForOp, convert them o be partitioned across workgroups or workitems.

The values for the workgroup/workitem id along each dimension is passed in with ids. The number of workgroups/workitems along each dimension are passed in with nids. The innermost loop is mapped to the x-dimension, followed by the next innermost loop to y-dimension, followed by z-dimension.
