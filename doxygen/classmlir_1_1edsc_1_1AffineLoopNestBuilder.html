Detailed Description

Explicit nested LoopBuilder.

Offers a compressed multi-loop builder to avoid explicitly writing all the loops in a nest. This simple functionality is also useful to write rank-agnostic custom ops.

Usage:

{c++}
AffineLoopNestBuilder({&i, &j, &k}, {lb, lb, lb}, {ub, ub, ub}, {1, 1,
1})(
[&](){
...
});
{c++}
AffineLoopNestBuilder({&i}, {lb}, {ub}, {1})([&](){
AffineLoopNestBuilder({&j}, {lb}, {ub}, {1})([&](){
AffineLoopNestBuilder({&k}, {lb}, {ub}, {1})([&](){
...
}),
}),
});
