MLIRContext *  context

static

Given MemRef sizes that are either static or dynamic, returns the canonical "contiguous" strides AffineExpr.

Strides are multiplicative and once a dynamic dimension is encountered, all canonical strides become dynamic and need to be encoded with a different symbol. For canonical strides expressions, the offset is always 0 and and fastest varying stride is always 1.

Examples:

  • memref<3x4x5xf32> has canonical stride expression 20*d0 + 5*d1 + d2.
  • memref<3x?x5xf32> has canonical stride expression s0*d0 + 5*d1 + d2.
  • memref<3x4x?xf32> has canonical stride expression s1*d0 + s0*d1 + d2.
