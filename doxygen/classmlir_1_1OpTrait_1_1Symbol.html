Detailed Description

template<typename ConcreteType>
class mlir::OpTrait::Symbol< ConcreteType >

A trait used to define a symbol that can be used on operations within a symbol table.

Operations using this trait must adhere to the following:

  • Have a StringAttr attribute named 'SymbolTable::getSymbolAttrName()'.
