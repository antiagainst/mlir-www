Detailed Description

A NamedAttributeList is used to manage a list of named attributes.

This provides simple interfaces for adding/removing/finding attributes from within a DictionaryAttr.

We assume there will be relatively few attributes on a given operation (maybe a dozen or so, but not hundreds or thousands) so we use linear searches for everything.
