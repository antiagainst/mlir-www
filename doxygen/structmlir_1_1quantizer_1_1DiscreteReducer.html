Detailed Description

template<typename T>
struct mlir::quantizer::DiscreteReducer< T >

A fact and reducer for tracking agreement of discrete values.

The value type consists of a |T| value and a flag indicating whether there is a conflict (in which case, the preserved value is arbitrary).
